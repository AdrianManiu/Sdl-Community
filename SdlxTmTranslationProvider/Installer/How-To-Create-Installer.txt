Instructions to create installer for SDLX TM provider plug-in.

1. Check you have Inno installed. If not, download installer from http... or use installer in InnoSetup5 folder

2. Copy the *signed* Sdl.Sdk.SdlxTmTranslationProvider.sdlplugin plug-in file to the installer folder

3. Open the .iss file (in installer folder) in Inno Setup

4. Check the file paths and set any necessary build numbers - currently these are TMS build numbers

5. Check the Studio major version number e.g. '10' in the install paths

6. Save changes to iss file

7. Click on the 'run' button to create the installer

The installer executable is created in the 'Output' subfolder.
