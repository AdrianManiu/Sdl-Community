TAUS Search
Helping people to find terms and phrases in the TAUS industry-shared translation corpora.
Powered by TAUS DATA

About

  • TAUS Search plugin for SDL Trados Studio was developed by Sdl Community

Technical Requirements

  • SDL Trados Studio 2009/2011/2014
  • Internet Connection

Setup

     Installation

  • Download the zip file from the SDL Open Exchange website
  • Double click on the .msi and follow the installation instructions.

     SDL Trados Studio

  • Start SDL Trados Studio
  • Check that the plugin is active from the Plug-ins dialog (Tools>Plug-ins)
  • Add the TAUS Search plugin to the list of ‘Translation Memories’ in the Project Settings.

Settings

     Connection

  • Specify your login name & password and then create a new authorization key
    A valid account with TAUS is required to create an authorization key; the authorization key never expires. 
    You can create a fee account following the instructions from the following web page: https://www.tausdata.org/index.php/component/users/?view=registration

     Filters

  • All segments maintained on the TDA TM have several attributes; these attributes can be used to filter your searches.
    They are: industry, content type, owner, product, provider; the attributes have numerical values, and each value is associated with a name for display.
    For full information on segment attributes, refer to the API documentation:     https://www.tausdata.org/apidoc

     Options

  • Ignore translated segments
    Does not search for segments that are already translated
  • Search timeout
    Ensures that the response time from the TAUS server does not exceed the timeout specified. The value can be set between 5 and 20 seconds.
